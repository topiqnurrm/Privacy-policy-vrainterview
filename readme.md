# Privacy Policy — VRAINTERVIEW

Halaman kebijakan privasi untuk aplikasi **VRAINTERVIEW** (Virtual Reality Interview Simulator).

## 🔗 Live Page
Halaman ini dapat diakses melalui GitHub Pages setelah diaktifkan.

## 📁 Struktur
```
├── index.html     # Halaman utama privacy policy
└── README.md      # Dokumentasi ini
```

## 🚀 Deploy via GitHub Pages
1. Buka repository di GitHub
2. Pergi ke **Settings → Pages**
3. Pilih branch `main` → folder `/ (root)`
4. Klik **Save**
5. URL akan tersedia di: `https://topiqnurrm.github.io/Privacy-policy-vrainterview/`
